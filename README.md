# Facial Emotion Recognition using MobileNetV3-Small

## Project Overview

This project implements a lightweight Facial Emotion Recognition (FER) system using MobileNetV3-Small and the FER2013 dataset.

The project is developed as part of a computer vision internship.

## Dataset

- FER2013

## Model

- MobileNetV3-Small
- PyTorch

## Features

- FER2013 preprocessing
- Data augmentation
- MobileNetV3-Small training
- Validation & testing
- ONNX export
- Lightweight model (<100 MB)

## Results

| Metric | Value |
|---------|-------|
| Train Accuracy | 75.9% |
| Validation Accuracy | 65.5% |
